# Stopwatch Web Application

A simple and interactive stopwatch web application built with HTML, CSS, and JavaScript. This app allows users to start, pause, reset, and record lap times in real-time. The app is designed to be user-friendly and can be easily integrated into various projects or used independently for basic time-tracking needs.

## Features:
- **Start, Pause, and Reset** functionalities.
- **Lap** time recording to track multiple intervals.
- Real-time time display in `hh:mm:ss` format.
- Clean and responsive design, optimized for both desktop and mobile devices.

## Technologies Used:
- HTML
- CSS
- JavaScript 

## How to Use:
1. Open `index.html` in your browser.
2. Use the buttons to control the stopwatch and track laps.
3. View lap times below the timer.
